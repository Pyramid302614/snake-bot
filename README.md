# Pull Requests (PRs)
I'm always open for pull requests! I made this code to be easily modded and changable, so if you wanted to you can:
- **Fork this repo** - Making a copy of Snake Bot and adding your own running environment and distributing it as a modded version of Snake Bot (Encouraged)
- **Change this repo** - Forking this repo, but then submitting a pull request to make your instance the actuall instance

**Design:** Every color, body of text, and little detail is carefully designed. Expect changes to your pull request or rejection if you are changing visible things!\
**Code:** All code in Snake Bot was intended to be easy to read and modify, but also to be fast and efficient. Once again, revisions and rejections can occur, but pretty unlikely for rejections if it's just a new utility or something. **Learn the code before changing it!**\


# What it's runs on
Snake Bot runs on a cheap VPS from https://webdock.io.
It's the ssd-pico4 server, which was so cheap (95 cents a month) that webdock removed it after an update
a little bit after I got it. Thank you 0xC0ffee1 for saving me money :pray:

The code runs on Node.JS, obviously, because it's Javascript, and uses the library called Discord.js, you may have
heard of it before. https://discordjs.org

In the VPS, I use Process Manager 2 (PM2, great thing) to keep it up and running.

And of course, this code does not run on it's own. It requires a seperate thing to actually connect to the `snakelet/adapter.js`.
It also needs a directory outside the snake-bot directory to store the database.

**If you want to make a modded version of Snake Bot, let me know, and ill help you set it up! 🙂**
